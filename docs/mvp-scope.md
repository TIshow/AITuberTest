# MVPスコープ定義

## MVPの定義

**「講演者がマイクで話しかけると、AIアバターが教育×AIテーマで音声応答する。すべてローカルで動く。」**

---

## MVPで必須の機能

### 対話コア

| 機能 | 説明 |
|---|---|
| マイク音声入力 | Macのマイクで講演者の音声を取得する |
| ローカルSTT | 音声 → テキスト変換（Whisper等、ネット不要） |
| RAG検索 | 入力テキストに関連する教育文書のチャンクを検索・取得 |
| ローカルLLM推論 | 取得したコンテキスト付きで回答テキストを生成（Ollama等） |
| 音声合成 | 回答テキスト → 音声（VOICEVOX、ローカル動作） |
| 音声再生 | 合成音声をスピーカーで再生する |
| リップシンク | 音声に合わせてVRMアバターの口が動く |

### アバター

| 機能 | 説明 |
|---|---|
| VRMアバター表示 | 画面にアバターが常時表示される |
| アイドルモーション | 待機中も自然に微動する |
| 基本表情制御 | 最低2種類（Neutral / Happy）。LLMの応答内容から制御できるとよい |

### 運用

| 機能 | 説明 |
|---|---|
| 完全ローカル動作 | STT / LLM / RAG / TTS / アバターすべてオフラインで動く |
| 状態表示 | 「待機中」「聞き取り中」「考え中」「話し中」が視覚的にわかる |
| 強制停止 | 応答途中でも即座に止められるボタン / キー |
| RAG文書の事前登録 | 講演者が用意したPDF・テキストをインデックス化して参照できる |

---

## MVPで不要 / 後回しにする機能

### 後回し（v1以降）

| 機能 | 理由 |
|---|---|
| 話題制限フィルタ | プロンプト設計で代替。精密なフィルタはv1で |
| 会話履歴の永続化 | 1セッション内の文脈保持で十分 |
| RAG文書のUI管理 | ファイル配置のみで十分（GUIは後回し） |
| 複数アバター切替 | 1キャラクターで十分 |
| スライド連動 | 対話が安定してから追加 |

### 意図的に除外（本質的に不要）

| 機能 | 理由 |
|---|---|
| 外部LLM（Gemini / OpenAI等） | 完全ローカル方針に反する |
| 固定台本の読み上げ | 対話形式に変更したため不要 |
| YouTube配信連携 | 講演用途には不要 |
| ライブコメント読み上げ | 不要 |
| テキスト入力フォーム | マイク入力のみで十分 |
| 多言語対応 | 日本語のみ |

## 将来拡張の候補

### v1

- 会話の文脈保持精度向上（マルチターン対話の質改善）
- RAG文書のGUI管理画面
- 話題制限の強化（RAGスコアが低い質問は「範囲外です」と答えるなど）
- スライド連動（特定のキーワードでスライドを進める等）

### v2

- LLMモデルの切替・チューニング機能
- 会話ログの記録・レビュー
- 感情・モーションの高度化（応答内容に応じた自動制御）
- 動画書き出し

## 段階的開発フェーズ

### Phase 0: 技術検証（最優先・実装前に完了する）

- [ ] AITuberKit のセットアップ・動作確認（VRM表示・リップシンク）
- [ ] VOICEVOX のローカルインストール・疎通確認
- [ ] Ollama のインストール・モデル選定・日本語応答品質確認
- [ ] Whisper（ローカルSTT）の精度・遅延確認（Mac環境）
- [ ] RAG基盤の選定（ChromaDB等）・動作確認
- [ ] 全コンポーネントを繋いだ End-to-End 疎通確認

### Phase 1: MVP

目標: 「本番で安心して使える対話システムを完成させる」

- [ ] マイク入力 → STT → LLM+RAG → VOICEVOX → アバター の一連フロー実装
- [ ] RAG用教育文書のインデックス化
- [ ] 状態表示UI
- [ ] 強制停止機能
- [ ] プロンプト設計（AI時代の教育テーマに絞る）
- [ ] ローカル完全動作の確認
- [ ] 本番リハーサル（3回以上エラーゼロ）

### Phase 2: v1

目標: 「対話品質と操作性を上げる」

- [ ] 文脈保持の改善
- [ ] RAG文書管理の改善
- [ ] スライド連動（任意）

---

## MVP 品質基準

本番投入前に以下をすべてクリアすること：

- [ ] マイクで質問 → 5秒以内に回答音声が出始める
- [ ] 教育×AIテーマの質問に的外れな回答をしない
- [ ] ネット未接続で全機能が動作する
- [ ] 強制停止が確実に動作する
- [ ] 5回連続試行してフリーズ・クラッシュゼロ
- [ ] 起動から対話可能状態まで5分以内（Ollamaモデルロード含む）
